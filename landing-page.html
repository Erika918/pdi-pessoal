Landing Page – Projeto de Apresentação Profissional

🎯 Objetivo do Projeto

Este projeto foi criado com o objetivo de aplicar os conhecimentos adquiridos em HTML e CSS, além de iniciar a prática com responsividade e estruturação visual de páginas web. A proposta é apresentar uma landing page simples, clara e funcional para uso como portfólio ou apresentação pessoal.

🛠️ Etapas de Desenvolvimento

  1. Criação da estrutura HTML com header, main e footer
  2. Aplicação de estilos com CSS externo
  3. Organização do conteúdo em seções: Sobre, Tecnologias e Contato
  4. Testes de responsividade em diferentes tamanhos de tela
  5. Publicação via GitHub Pages

📐 Boas Práticas Aplicadas

  • Uso de tags semânticas como <header>, <main>, <section>, <footer>
  • Separação entre estrutura (HTML) e estilo (CSS)
  • Uso de classes para estilização em vez de estilos inline
  • Indentação e organização do código para melhor leitura

🧪 Testes de Responsividade

O layout foi testado em dispositivos móveis e monitores maiores, utilizando a ferramenta de inspeção do navegador para simular diferentes tamanhos de tela. Também foram aplicadas media queries simples para ajustar o espaçamento e fontes.
